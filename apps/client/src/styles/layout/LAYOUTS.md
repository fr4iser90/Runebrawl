# Runebrawl Layout ASCII

## 1) Global Game Shell (`shell.css`)

```text
+----------------------------------------------------------------------------------+
| TOP HUD                                                                          |
| [round] [gold] [hp] [xp] [actions...]                                            |
+-------------------------------------+--------------------------------------------+
| CENTER STAGE                        | RIGHT RAIL                                 |
| (phase view content)                | [players]                                  |
|                                     | [log]                                      |
|                                     |                                            |
+-------------------------------------+--------------------------------------------+
| BOTTOM BAR                                                                       |
| [tavern/combat controls/context actions]                                         |
+----------------------------------------------------------------------------------+
```

```text
+-------------------------------------------------------------------------------+
| HALL HEADER                [status/economy]                                   |
|-------------------------------------------------------------------------------|
| HERO ROW (optional)                                                           |
| [hero card] [hero card] [hero card]                                           |
|-------------------------------------------------------------------------------|
| SHOP ROW                                                                      |
| [unit] [unit] [unit] [unit] [unit] ...                                        |
|                                                                               |
| density modes:                                                                |
| - roomy:    [  big cards  ]                                                   |
| - balanced: [ medium cards ]                                                  |
| - dense:    [ small cards  ]                                                  |
| - stacked:  [][ ][ ][ ][ ]  (overlap/fan)                                     |
|-------------------------------------------------------------------------------|
| HALL ACTIONS                                                                  |
| [roll] [freeze] [upgrade] [ready]                                             |
+-------------------------------------------------------------------------------+
```
